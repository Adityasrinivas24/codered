Problem Statement CR 02



Blockchain Based Aircraft Maintenance

Blockchain-based aircraft maintenance aims to securely log maintenance records on a blockchain. Storing date-stamped and digitally signed service details by responsible parties ensures an immutable record. This record enhances aircraft value during resale and aids in error identification during undesired events. Create a user interface to streamline data entry, enabling authorized personnel to input maintenance activities easily and access the blockchain-stored records for verification and auditing purposes.

Problem Statement CR 03



Pose Recommendation System

The Pose Recommender System is an innovative machine learning application designed to recommend optimal poses tailored to individual preferences and characteristics. Your model should be able to accurately detect body keypoints from images enabling precise analysis of various factors but not limited to outfit, body shape, and background. Your model should be able to gauge pose similarity and diversity to suggest the most suitable pose. The user interface empowers individuals to input their media and preferences, receiving tailored pose recommendations while offering guidance to refine their posture.

Problem Statement CR 04



Smart Price Comparison Engine

Create a web application or a mobile application with smart engine that can fetch and compare the prices of the same product from multiple e-commerce websites, such as Amazon, Flipkart, Snapdeal, etc. The engine should also be able to provide additional information, such as ratings, reviews, delivery options, discounts, etc. The engine should have features such as web scraping, data analysis, and user interface.
